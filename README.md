Bank Transaction Management System

A simple Java + JDBC + Oracle Database based Bank Transaction System that allows:
Account Validation
Balance Checking
Money Transfer Between Accounts
Transaction History Storage

This project follows a layered architecture:
Bean Layer
DAO Layer
Service Layer
Utility Layer
Main Application Layer

Technologies Used
Java
JDBC
Oracle Database
SQL
Eclipse IDE
